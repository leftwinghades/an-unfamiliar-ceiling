Prototype of my website using Nodejs as the backend, PostgreSQL as the database.

Sign up:
1. Checks if the username and password meet the requirement.
2. Sends an sql query to the database to see if the username has been taken.
3. Hashes the password and then sends an sql query to the database to store the username and the hashed password.

Log in:
1. Checks if the username and password meet the requirement.
2. Sends an sql query to see if the username is in the database.
3. Checks if the entered password hashed matches the one in the database.
4. Creates a JSON Web Token and stores it in cookies.

Accessing this page:
1. Sends a get request to this route with the JSON Web Token.
2. Verifies the JSON Web Token to see if it is valid then proceeds if it is.

Log out:
1. Deletes the JSON Web Token stored in the cookies.
2. Redirects to homepage.
